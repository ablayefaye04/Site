Langages de Programmation

Je maîtrise plusieurs langages de programmation, allant du développement bas-niveau au développement orienté objet et au scripting pour diverses plateformes :

  • Java : Mon langage principal pour le développement d'applications backend robustes, en particulier pour les systèmes d'entreprise et les applications Android.
  • C : Utilisé pour des applications à bas-niveau et la programmation système, où les performances sont critiques.
  • Python : Idéal pour le scripting, les applications web avec des frameworks comme Django, ainsi que l'apprentissage automatique avec NumPy et TensorFlow.
  • JavaScript : Utilisé pour le développement front-end interactif avec HTML5 et CSS3. Je l'utilise également avec des frameworks comme React pour créer des interfaces dynamiques.
  • HTML/CSS : Les bases du développement web, que j'utilise pour structurer et styliser mes pages web, souvent avec Bootstrap pour une mise en page responsive.

Frameworks et Bibliothèques

J'ai une expérience approfondie avec Jakarta EE, un ensemble de spécifications pour le développement d'applications d'entreprise évolutives et robustes. Cela inclut la gestion des servlets, JSP, et l'injection de dépendances via CDI.

Base de Données

J'utilise MySQL pour gérer les bases de données relationnelles. Je suis à l'aise avec la conception de schémas, l'optimisation des requêtes SQL, ainsi que l'intégration avec des applications web via JDBC ou JPA.

Développement Web

Mon expertise en développement web inclut l'utilisation de JSP, Servlets, HTML5, CSS3 et JavaScript pour créer des interfaces utilisateur dynamiques et des systèmes backend robustes. Je combine souvent ces technologies avec Bootstrap pour une mise en page responsive.

Développement Mobile

J'ai de l'expérience dans la création d'applications mobiles pour Android, en utilisant Android Studio et Java. Je suis à l'aise avec la création d'interfaces utilisateur réactives ainsi qu'avec l'intégration de bases de données locales (SQLite).

Systèmes d'exploitation

Je travaille aussi bien sous Linux que sous Windows. J'utilise Linux pour le développement et la gestion de serveurs, et Windows pour le développement d'applications spécifiques et la gestion de l'infrastructure Windows Server.
